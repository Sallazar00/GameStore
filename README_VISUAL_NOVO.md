# GameStore — front-end renovado e loja digital

O projeto utiliza um visual responsivo de loja de games e agora funciona exclusivamente com produtos digitais.

## Destaques

- Cabeçalho responsivo com busca, carrinho, conta e administração.
- Página inicial com banner, categorias, filtros e cards de jogos.
- Produto, carrinho, checkout, login, cadastro e biblioteca padronizados.
- Painel administrativo para produtos, categorias e plataformas.
- Checkout integrado ao CaçaPay, sem endereço ou logística.
- CSS principal em `public/css/store.css`.
- Bootstrap armazenado localmente.

Consulte `README.md` e `README_CACAPAY_DIGITAL.md` para instalação e configuração da API.
